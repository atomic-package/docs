# Footer

<p class="uk-text-lead">Easily create nice looking footer, which come in different styles.</p>

```html
<footer class="footer">
  <div class="top">
    <div class="wrap">Footer Layout Top</div>
  </div>
  <div class="main">
    <div class="wrap">Footer Layout Main</div>
  </div>
  <div class="bottom">
    <div class="wrap">Footer Layout Bottom</div>
  </div>
</footer>
```


## Color Variation

The color classes defined in the atomic package can be adapted.

```html
  <div class="main">
    <div class="wrap">
      <p class="subTitle footer">APBCSS UI Kit for everyone to share and enjoy.</p>
    </div>
  </div>
</footer>
```
