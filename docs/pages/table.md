# Table

<p class="text docsLeadText">Easily create nice looking tables, which come in different styles.</p>

## Usage

To apply this component, add the `.table` class to a `<table>` element.

```html
<table class="table">
  <caption>Table caption</caption>
  <thead>
  <tr>
    <th>Table Heading</th>
  </tr>
  </thead>
  <tfoot>
  <tr>
    <td>Table Footer</td>
  </tr>
  </tfoot>
  <tbody>
  <tr>
    <td>Table Data</td>
  </tr>
  <tr>
    <td>Table Data</td>
  </tr>
  <tr>
    <td>Table Data</td>
  </tr>
  </tbody>
</table>
```

## Striped modifier

Add zebra-striping to a table by adding the `.striped` class.

```html
<table class="table striped">...</table>
```

```example
<table class="table striped">
  <caption>Table caption</caption>
  <thead>
  <tr>
    <th>Table Heading</th>
    <th>Table Heading</th>
    <th>Table Heading</th>
  </tr>
  </thead>
  <tfoot>
```
