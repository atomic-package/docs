# List

<p class="text docsLeadText">Easily create nice looking lists, which come in different styles.</p>

## Usage

You can apply this component by adding the `.list` class, `atomic class`, to the list of ordered or unordered classes. Now the list will be displayed without any spaces or list styles.

```html
<ul class="list">
    <li></li>
    <li></li>
    <li></li>
</ul>
```

```example
<ul class="list">
    <li>List item 1</li>
    <li>List item 2</li>
    <li>List item 3</li>
</ul>
```

## Bullet modifier

Add the `.circle` class to display bullets in front of the list items.

```html
<ul class="list circle">
    <li>List circle</li>
</ul>
```

```example
<ul class="list circle">
    <li>List circle</li>
</ul>
```
