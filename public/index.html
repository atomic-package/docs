    Features

    Atomic-PackageはあなたのHTML、CSS、およびJavaScriptコンポーネントの包括的なコレクションを提供します。
    これはカスタマイズが容易で拡張、使用するのは簡単です。

    • Atomic Design
      APB CSS

      APBCSSをベースに作成されたパッケージなので、APBCSSのアーキテクチャを容易に導入することが可能です。

    • UI Parts
      Components

      汎用性のある、UI Partsのコンポーネントが用意されています。

    • Customize

      テーマファイルを独自に作成することによって、カスタマイズも容易におこなえます。

    • Responsive

      レスポンシブデザインにも対応しているので、スマートフォンサイトでも利用が可能です。
